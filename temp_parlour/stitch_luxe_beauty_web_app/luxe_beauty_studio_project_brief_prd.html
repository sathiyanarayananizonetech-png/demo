# Luxe Beauty Studio 🌿✨ A premium, high-performance web application for a luxury beauty salon. ## 🎨 Design System ### Color Palette - **Primary Gold**: `#C9A24A` (Brand identity, primary buttons, accents) - **Gold Container**: `#E5C98E` (Lighter gold for containers and hover states) - **Secondary Cream**: `#D9C3A5` (Supporting elements and soft borders) - **Tertiary Navy**: `#1F3A5F` (Deep contrast for dark modes or secondary sections) - **Background**: `#F4F1EC` (Main page background - Luxury Cream) - **Charcoal Slate**: `#2B2B2B` (Main typography and heavy backgrounds) ### Typography - **Primary Heading**: `Poppins` (Bold & Commanding) - **Secondary Heading**: `Montserrat` (Elegant & Modern) - **Body Text**: `Inter` (Clean & Readable) - **Decorative**: `Pacifico` & `Gelasio` (Italic & Script accents) ## 🧩 Key Components - **ServiceCard**: Hover-reveal descriptions, image scaling, integrated price tag. - **ShuffleGrid / Hero**: Dynamic shuffling image grid with spring-based animations. - **GalleryChapters**: Storytelling section with snap-to-section scrolling and background grid patterns. ## 🗺️ Project Structure & Tech - React 19, Vite, Tailwind CSS v4, GSAP, Framer Motion, Lenis, Lucide React.
